# Virtual Cell Challenge

# Introduction
Exploratory analysis for the Virtual Cell Challenge using PROPD and other metrics.


## Repository Structure 
VCC/
├─ data/             Input data used by analyses.  
├─ results/          Outputs generated by the analyses.  
├─ plots/            Figures and visual material.  
├─ scripts/          Executable scripts that run analyses.  
├─ vcc/              Reusable Python package with core logic.  
│  ├─ config.py      Central location for common paths and shared settings.  
│  ├─ analysis/      Functions and modules used by scripts.  
│  └─ plotting/      Plotting functions and utilities.  
└─ README.md         Project documentation.
